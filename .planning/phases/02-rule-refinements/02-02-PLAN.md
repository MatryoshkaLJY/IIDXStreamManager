## Task 3: Update Sorting Rules and Tie-Breaker Logic

<task>
  <name>Implement Group and Finals Sorting with Tie-Breaker</name>
  <type>auto</type>
  <files>
    - iidx_knockout_scoreboard/app.js
  </files>
  <read_first>
    - iidx_knockout_scoreboard/app.js
  </read_first>
  <action>
    1. In `handleSettle()` for non-final stages (quarterfinal, semifinal/E/F):
       - Keep the existing sort: `b.points - a.points` then `b.totalRawScore - a.totalRawScore`.
    2. In `handleSettle()` for finals (`stage === 'final'`):
       - Sort by `points` descending only.
       - Check if there is a tie for 1st place: `if (sortedPlayers[0].points === sortedPlayers[1].points)`.
       - If tied:
         - Set `this.tournamentState.finals.tiebreaker = true`.
         - Set `this.tournamentState.finals.settled = false`.
         - Log: ` console.log('🔄 Finals tie detected — extra round required');`
         - Do NOT eliminate/advance; leave all 4 players in current state.
         - Return early from handleSettle.
       - If NOT tied:
         - `this.tournamentState.finals.tiebreaker = false`.
         - Proceed to normal final settlement and medal display.
    3. During a tie-breaker extra round (`tiebreaker === true`):
       - In `handleScore()`, when `stage === 'final'`, check `if (this.tournamentState.finals.tiebreaker)`.
       - If true, accumulate `rawScores` and update `totalRawScore`, but skip adding PT points (do not call `calculatePoints` or add to `player.points`).
       - Update the node score display with `totalRawScore`.
    4. When `handleSettle()` is called again for finals while `tiebreaker === true`:
       - Sort by `totalRawScore` descending.
       - The top player is champion. Clear `tiebreaker = false`; set `settled = true`.
       - Proceed to medal display.
  </action>
  <verify>
    - Non-final stages sort by points then totalRawScore
    - Finals sort by points descending only (before tie-breaker check)
    - Finals tie condition checks `sortedPlayers[0].points === sortedPlayers[1].points` and sets `tiebreaker = true`
    - `handleScore` for finals has a branch: `if (this.tournamentState.finals.tiebreaker) { ... }` that skips point awards
    - Tie-breaker settle branch sorts by `totalRawScore` descending and declares champion
  </verify>
  <done>
    Group sorting, finals PT-only sorting, and tie-breaker extra rounds are fully implemented.
  </done>
</task>

    1. `from flask import Flask, render_template`

         - Attempts `load_configs()`

       - `app = create_app()`
       - `socketio.run(app, host="0.0.0.0", port=5002)`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser → Flask app | Untrusted HTTP requests to local operator UI |
| filesystem → state.py | Local state file read/write |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information Disclosure | `SECRET_KEY` fallback | mitigate | Use `os.environ.get("SECRET_KEY", "dev-secret-change-in-production")` — never hardcode a production secret; document that operators should set `SECRET_KEY` for production (ASVS L1) |
| T-02-02 | Tampering | `runtime/state.json` | accept | Local filesystem only; no remote access |
| T-02-03 | Denial of Service | Large state file | mitigate | State file is bounded (only config paths + timestamp); no user-controlled data is written without validation |
</threat_model>

# Phase 1 — UI Design Contract

> Visual and interaction contract for Phase 1: Foundation & Config. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |
| Component library | none (plain HTML + minimal CSS) |
| Icon library | none (text labels preferred for operator clarity) |
| Font | system-ui, sans-serif |

Notes:
- This is a Flask server-rendered Jinja2 application. No React, no shadcn, no component library.
- Phase 1 UI is intentionally minimal: a functional operator control panel with a status dashboard, config validation messages, and basic navigation.
- Desktop browser only; no mobile responsiveness required per REQUIREMENTS.md.

---

## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Inline element gaps, tight padding |
| sm | 8px | Compact row spacing, small buttons |
| md | 16px | Default card padding, form gaps |
| lg | 24px | Section separation |
| xl | 32px | Major layout gaps |
| 2xl | 48px | Page-level vertical rhythm |

Exceptions: none

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 | 1.5 |
| Label | 14px | 600 | 1.4 |
| Heading | 20px | 600 | 1.2 |
| Display | 28px | 700 | 1.15 |

Notes:
- Use `system-ui` font stack for crisp rendering on Windows streaming PCs.
- Monospace (14px / 400 / 1.4) reserved for file paths and raw error details.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0f172a | Page background, primary surfaces |
| Secondary (30%) | #1e293b | Cards, panels, nav bar |
| Accent (10%) | #22c55e | Success indicators, active status, valid config |
| Destructive | #ef4444 | Validation errors, critical failures |
| Warning | #f59e0b | Non-blocking issues, pending states |
| Text primary | #f8fafc | Headings, primary labels |
| Text secondary | #94a3b8 | Body text, metadata, file paths |

Accent reserved for:
- Application running / server online indicator
- Config loaded successfully message
- Valid schema checkmark

Destructive reserved for:
- JSON validation errors
- Missing file errors
- Fatal startup issues

## Copywriting Contract

|---------|------|

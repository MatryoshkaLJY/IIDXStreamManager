       ```python
       @app.route("/monitor_control", methods=["POST"])
       def monitor_control():
           data = request.get_json(silent=True) or request.form.to_dict()
           action = (data.get("action") or "").strip().lower()
           if action not in ("start", "stop"):
               return jsonify({"success": False, "error": "Invalid action. Use 'start' or 'stop'."}), 200

           runtime_state = load_runtime_state()
           monitor = app._cabinet_monitor

           if action == "start":
               runtime_state.monitoring_active = True
               save_runtime_state(runtime_state)
               monitor.start()
           else:
               runtime_state.monitoring_active = False
               save_runtime_state(runtime_state)
               monitor.stop()

           socketio.emit("monitoring_status", {"active": runtime_state.monitoring_active})
           return jsonify({"success": True, "active": runtime_state.monitoring_active}), 200
       ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser -> Flask API | Untrusted HTTP POST crosses into the application |
| Flask API -> background thread | Monitor control actions cross into the CabinetMonitor worker thread |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-03-01 | Tampering | /monitor_control endpoint | mitigate | Strictly validate action is only "start" or "stop"; reject all other values with 200 + JSON error |
| T-05-03-02 | Denial of Service | /monitor_control rapid toggles | mitigate | RuntimeState persists immediately; no debounce needed for a single-operator local UI, but the backend interval clamp in CabinetMonitor prevents runaway resource usage |
| T-05-03-03 | Information Disclosure | cabinet_update SocketIO event | accept | Data is intended for the operator UI; no PII or secrets are emitted |
| T-05-03-04 | Repudiation | monitoring_active state | accept | State is persisted to runtime/state.json; operator can verify current state on disk |
</threat_model>

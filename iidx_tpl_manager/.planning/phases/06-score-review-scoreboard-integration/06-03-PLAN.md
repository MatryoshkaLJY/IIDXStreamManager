    Extend static/js/operator.js with the following functionality:

    1. **DOM references** (at the top of the IIFE):
       ```javascript
       const reviewPanel = document.getElementById('score-review-panel');
       const reviewRows = document.getElementById('score-review-rows');
       const reviewToggle = document.getElementById('score-review-toggle');
       const confirmBtn = document.getElementById('confirm-score-btn');
       const countdownEl = document.getElementById('score-countdown');
       const delayInput = document.getElementById('scoreboard-delay');
       const reviewError = document.getElementById('score-review-error');
       const reviewEmpty = document.getElementById('score-review-empty');
       ```

    2. **State**:
       ```javascript
       let pendingScores = window.initialPendingScores || {};
       let scoreboardDelay = window.initialScoreboardDelay || 5.0;
       let countdownInterval = null;
       let isCollapsed = false;
       ```

    3. **Render function** `renderReviewPanel()`:
       - Clear reviewRows.innerHTML.
       - If pendingScores is empty, show reviewEmpty and hide confirmBtn and reviewError.
       - Otherwise, hide reviewEmpty, show confirmBtn, and build a row for each machine_id in ["IIDX#1","IIDX#2","IIDX#3","IIDX#4"] that exists in pendingScores.
       - For each row, create the grid with: machine label, player name (from a data attribute or passed server-side), 1P score cell, 2P score cell, validity indicator.
       - Apply `score-invalid` class to a score cell if its valid flag is false.
       - Show `reviewError` if any score is invalid, and disable confirmBtn.
       - If the panel was auto-expanded due to new scores, ensure it is visible.

    4. **Inline editing**:
       - Add a delegated click listener on reviewRows for `.score-cell` elements.
       - On click, replace the cell text with an `<input type="number">` pre-filled with the current value.
       - On blur or Enter key, commit the edit: update `pendingScores[machine_id].scores[side + "_score"]` and re-render.

    5. **Toggle collapse**:
       - If reviewToggle exists, add click listener to toggle `isCollapsed` and update reviewRows.style.display and button text ("Hide" / "Show").

    6. **Scoreboard delay save**:
       - Add blur and keydown listeners to delayInput.
       - On blur or Enter, POST JSON to `/api/scoreboard_delay` with `{delay: parseFloat(delayInput.value)}`.

    7. **Confirm & Push**:
       - Add click listener to confirmBtn.
       - On click, if any score is invalid, do nothing (button is already disabled).
       - Otherwise, POST JSON to `/confirm_score` with empty body.
       - On success response, start countdown:
         - Show countdownEl.
         - Set text to `Pushing in ${Math.ceil(scoreboardDelay)}s...`.
         - Use setInterval to decrement every second.
         - At 0, change text to "Pushing scores..." and add `pulsing` class.
         - After 1 more second, clear interval, hide countdownEl, and clear reviewRows (pendingScores = {}).

    8. **SocketIO integration**:
       - Update the existing `socket.on('cabinet_update', ...)` handler to also update pendingScores:
         ```javascript
         socket.on('cabinet_update', (data) => {
           console.log('cabinet_update', data);
           if (data.state === 'score' && data.score_validation_pending) {
             pendingScores[data.machine_id] = {
               machine_id: data.machine_id,
               scores: data.scores || {},
             };
             renderReviewPanel();
           } else if (data.state !== 'score' && pendingScores[data.machine_id]) {
             delete pendingScores[data.machine_id];
             renderReviewPanel();
           }
         });
         ```
       - Add handler for `scores_pushed`:
         ```javascript
         socket.on('scores_pushed', () => {
           pendingScores = {};
           renderReviewPanel();
           if (countdownInterval) {
             clearInterval(countdownInterval);
             countdownInterval = null;
           }
           countdownEl.style.display = 'none';
           countdownEl.classList.remove('pulsing');
         });
         ```
       - Add handler for `scoreboard_delay_updated`:
         ```javascript
         socket.on('scoreboard_delay_updated', (data) => {
           if (typeof data.delay === 'number') {
             scoreboardDelay = data.delay;
             if (delayInput) delayInput.value = scoreboardDelay;
           }
         });
         ```

    9. **Initial render**:
       - Call `renderReviewPanel()` at the end of the IIFE.
